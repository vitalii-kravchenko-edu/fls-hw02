Основи HTML: Структура та Основні Теги

Вступ

HTML (HyperText Markup Language) - це мова розмітки, яка визначає структуру веб-сторінок. Основою HTML є теги, які використовуються для побудови елементів сторінки. Вони можуть бути як блочними, так і рядковими. Документ, розмічений за допомогою HTML, інтерпретується браузером. Як результат, користувачі бачать не вихідний код з елементами розмітки, а результат обробки цього файлу — веб-сторінку.

Основи синтаксису HTML

  1. Основна структура документа

    Кожен HTML-документ починається з оголошення <!DOCTYPE html>, яке вказує на тип документа. Далі йде <html> – кореневий тег, що містить дві основні частини:

    • <head> – містить службову інформацію (мета-дані, назву сторінки тощо).

    • <body> – містить весь видимий контент сторінки.

  2. Правила будови HTML-документа:

    • Теги можуть містити атрибути.
    • Усі відкриті теги повинні закриватися.
    • Деякі теги є самозакритими.

Основні HTML-теги

  1. Будівельні теги

    Будівельні теги використовуються для створення загальної структури сторінки:

    • <header> – верхня частина сайту або секції.
    • <main> – основний контент сторінки (повинен бути лише один).
    • <footer> – нижня частина сайту або секції.
    • <div> – контейнер без конкретного призначення, використовується для групування елементів.

  2. Контентні теги

    Контентні теги впливають на вміст сторінки:

    • Заголовки: <h1> – <h6>, де <h1> – головний заголовок (лише один на сторінці).
    • Абзаци: <p> – використовується для текстових блоків.
    • Виділення тексту:
      • <span> – виділення частини тексту без зміни вигляду.
      • <mark> – підсвічування важливих фрагментів тексту.
      • <b> – виділення фрагментів тексту жирним шрифтом.
      • <code> – виділення коротких <фрагментів коду>.
    • Розрив рядка: <br> – переносить текст на новий рядок.

  3. Списки

    HTML підтримує два основних типи списків:

    • Маркований список (<ul>)

    • Нумерований список (<ol>)

    Списки можуть містити вкладені елементи, кожен з яких огортається тегом <li>.

  4. Зображення та посилання

    • Зображення:

      Де src – шлях до файлу, а alt – альтернативний текст.

    • Гіперпосилання:

      Де href – шлях для переходу. Додатково можна використовувати атрибут target="_blank" для відкриття посилання в новій вкладці.

Властивості елементів

  1. Атрибути

    Атрибути додають тегам додаткові властивості. Основні з них:

    • class – дозволяє застосовувати стилі до групи елементів.
    • id – унікальний ідентифікатор елемента.
    • title – підказка, що з’являється при наведенні мишки.

  2. Блочні, рядкові та блочно-рядкові елементи

    • Блочні: займають всю ширину батьківського контейнера (усі будівельні: <div>, <header>, <main>, <footer> та інші, а також контентні: <h1> - <h6>, <p>, <ul>, <ol>, <li>).
    • Рядкові: займають ширину контенту (наприклад, <span>, <a>, <mark>, <br>).
    • Блочно-рядкові: поєднують обидві властивості (наприклад, <img>).

Висновок

HTML є основою будь-якого веб-сайту. Знання його структури, основних тегів та атрибутів допомагає правильно верстати сторінки та організовувати контент.
